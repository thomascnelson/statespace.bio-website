Biology as Dynamical Systems

We work on projects that study the dynamics of biological state space to identify the potential interventions that could move a system from one state to another.

Explore the Idea

The Dynamic System Approach

In simple terms, a state space is a mathematical topology of all possible conditions in which a system can exist. Every point in this space represents a specific "state" defined by a set of variables. What defines a State Space? To build a state space, you first identify the variables that change over time. These variables become the axes of your map. Dimensions: If you are tracking 2 variables (e.g., oxygen levels and heart rate), your state space is 2D. If you track 10,000 genes, it’s a 10,000-dimensional space. Coordinates: A single point in this space tells you exactly what the system is doing at one moment in time. Trajectories: As the system changes, the point moves, drawing a line called a trajectory. Biology is messy and dynamic, which makes state space an incredible tool for finding order in the chaos. Why does it matter? State space allows biologists to move from just describing what is happening to predicting what will happen next. By looking at the topology (the shape) of the state space, we can identify things like stability (will the system return to normal after a disturbance), bifurcations (at what point does a small change like a slightly higher temperature cause a massive, irreversible shift in an ecosystem) and control (can we "nudge" a cancer cell out of its growth state and into a "cell death" state by changing its coordinates in the state space). It’s especially useful when biology becomes too complex to understand by looking at one variable at a time.

Potential Application

Realistic Expectations

We need to explicitly distinguish between candidates and causal drivers, we are aiming for a high-precision hypothesis generation engine.

Long-Term Vision

If biological states are able to be modeled, then cellular and organismal states are theoretically redirectable toward desirable states. Our core mission is to determine if this is possible and if so, solve for the perturbations needed to move a cell or organism from state A to state B.
